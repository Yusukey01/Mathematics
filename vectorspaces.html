Vector Space

A vector space is a nonempty set \(V\) of vectors on which are defined additionand scalar multiplication subject to the ten axioms.
  1. \(u + v \in V\)
  2. \(u + v = v + u\)
  3. \((u + v) + w = u + (v + w)\)
  4. \(0 \in V\) s.t. \(u + 0 = u\)
  5. \(\forall u \in V, \exists -u\) s.t. \(u + (-u) = 0\)
  6. \(cu \in V\)
  7. \(c(u + v) = cu + cv\)
  8. \((c + d)u = cu + du\)
  9. \(c(du)=(cd)u\)
  10. \(1u = u\)
