Symmetric Matrices

A symmetric matrix is a square matrix \(A\) such that \[A^T= A\] Remember, we learned a least-squares solution \(\hat{\beta}\) satisfies \(X^TX\beta = X^TY\). In this expression, \(X^TX\) is always symmetric. Let X be an \(m \times n\) matrix. Then \[(X^TX)^T = X^T(X^T)^T = X^TX.\]
An \(n \times n\) matrix \(A\) is orthogonally diagonalizable if there an orthogonal matrix \(P\) and a diagonal matrix \(D\) such that \[A = PDP^T = PDP^{-1}\] So, we need \(n\) linearly independent and orthonormal eigenvectors for this diagonalization. In this special case, \(A\) is actually symmetric becuase \[A^T = (PDP^T)^T = (P^T)^T D^T P^T = PDP^T = A \]
