Gamma Function(Bonus)

The Gamma function is defined as: \[ \Gamma (z) = \int_{0} ^\infty t^{z-1}e^{-t} dt \] where \(z\) is any complex number with postive real part.

Also, for all positive integers \(n\), \[ \Gamma (n) = (n-1)! \text{ , or equivalently, } n! = \Gamma (n+1) \] Substitute \(z= n+1\) \[ \Gamma (n+1) = \int_{0} ^\infty t^{n}e^{-t} dt \] When \(n =1\): \[ \Gamma (2) = \int_{0} ^\infty te^{-t} dt \] By integration by parts
(\(u = t \Longrightarrow du =dt\), and \(dv = e^{-t} dt \Longrightarrow v = -e^{-t}\)) \[ \begin{align*} \Gamma (2) &= [-te^{-t}]_{0} ^\infty + \int_{0} ^\infty e^{-t} dt \\\\ &= 1 \end{align*} \] Next, we assum that \(\Gamma(k+1) = k!\) for some \(k \in \mathbb{Z}^+\) : \[ \Gamma (k+1) = \int_{0} ^\infty t^{k}e^{-t} dt = k! \] Using the recursive property of the Gamma function: \[ \Gamma (k+2) = (k+1)\Gamma(k+1) = (k+1)k! = (k+1)! \] Therefore, \[ \Gamma(n+1) = n! \]
