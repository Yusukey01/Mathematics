Discrete Random Variable

In machine learning, we can consider any unknown quantities as random variables.

Let \(x\) be a discrete random variable and \(f(x) = P(X = x)\). Then
  1. \( 0 \leq f(x) \leq 1\) for \(i = 1, 2, \cdots, k\).
  2. \(\sum_{i =1}^k f(x_i) = 1\).
  3. \(P(x \in A) = \sum_{x \in A} f(x)\).
and we call \(f(x)\) a probability mass function (pmf).

We define some important quantities:
  1. Expected value (mean):\(\mu = \mathbb{E}[X] = \sum_{i}x_if(x_i) \)
  2. Variance: \(\sigma^2 = \text{ Var }(X) = \mathbb{E}[(x - \mu)^2]\)
  3. Standard deviation: \(\sigma = \sqrt{\text{Var }(X)}\)
