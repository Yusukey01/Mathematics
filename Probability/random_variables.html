Discrete Random Variable

In machine learning, we can consider any unknown quantities as random variables. A random variable associates a distinct numerical value with each outcome in the sample space. (Formally, it is a single valued fuction.) Usually, we denote a random variable by a capital letter(\(X\)) and a specific value taken by a random variable by the corresponding lower case letter\(x\).

A random variable is discrete if the number of possible values it takes is finite or countably infinite. We can describe the collection of probabilities as a function of \(x\): \[ f(x) = P(X = x) \] we call \(f(x)\) a probability mass function (p.m.f.).
Then
  1. \(f(x) \geq 0\) for all \(x\).
  2. \(\sum_{x} f(x) = 1\).
  3. Cumulative distribution function(c.d.f.): \(F(x) = P(X \leq x) = \sum_{k \leq x} f(k)\).

We define some important quantities:
  1. Expected value (mean):\(\mu = \mathbb{E}[X] = \sum_{i}x_if(x_i) \)
  2. Variance: \(\sigma^2 = \text{ Var }(X) = \mathbb{E}[(x - \mu)^2] \geq 0\)
  3. Standard deviation: \(\sigma = \sqrt{\text{Var }(X)}\)
