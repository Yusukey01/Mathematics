Probability

The collection of every possible outcome of an experiment is called a sample space denoted as \(S\). It can be discrete or continuous. An event \(A\) is a set of outcomes of an experiment, or a subset of the sample space \(A \subseteq S\). The probability of \(A\) denoted as \(P(A)\) satisfies the following axioms:
  1. \(P(A) \geq 0\)
  2. \(P(S) = 1\)
  3. If the events \(A\) and \(B\) are mutually exclusive, \(P(A \cup B) = P(A) + P(B)\)

Using event algebra, the follwing basic facts can be derived:
  1. Complement: \(P(A^c) = 1 - P(A)\)
  2. \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)
  3. \(P(A) = P(A \cap B) + (A \cap B^c)\)
  4. If \(B \subset A\), then \(A \cap B =B\), and so \(P(A) - P(B) = P(A \cap B^c)\)
  5. \(P[(A \cup B)^c] = P( A^c \cap B^c)\)
  6. \(P[(A \cap B)^c] = P( A^c \cup B^c)\)
