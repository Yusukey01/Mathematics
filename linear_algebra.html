Linear Algebra

Linear Algebra is one of the most fundamental area of modern mathematics. Probably, all scientists and engineers are appreiciate this powerful tool. In linear algebra, understanding the concepts is more important than the computations because in practice, nobody can handle large scale computations without a computer. Note that we can consider linear algebra as a part of the modern algebra aka abstract algebra, which provides each branch of mathematics with "common language." If you love pure mathematics, you must learn it.
